# Roboto (2013 Version)
The 2013 version of Roboto.

# Info
This is a version of Roboto from back in 2013. It differs from the 2011 version by making the "tail" from the comma less straight, makes the glyphs a bit more refined, fixes **most** mistakes of the glyphs (most fixes happened on the lighter weights) and adds more characters. This release places it between the font created date of 2011 and the major redesign in 2014.

The fonts are licensed under the Apache 2.0 license.

Article about the differences between the 2011 and 2013 versions: https://www.androidpolice.com/2013/06/29/typeface-teardown-roboto-gets-a-facelift-in-android-4-3/
